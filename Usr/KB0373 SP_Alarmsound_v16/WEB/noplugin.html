Welcome to Webvue !

The Java Plug-in has not been detected on your computer.
That component is required in order to run WebVue.
You need to install it before the first use.

The Java Plugin is available:

  • On the supervisor CD-ROM, in the directory \\JavaPlugin
  • On download on the WebVue Server
  • On download on Sun's Web Site

When the Java Plug-in is installed, please click on the Refresh button.

If you are sill experiencing problems, please read the online help.
